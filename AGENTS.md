# Tele Drive - TDLib Information

This project is pinned to **TDLib v1.8.0**.

## Requirements
- **TDLib Version:** 1.8.0
- **Java Bindings:** Official generated bindings for v1.8.0 are used in `org.drinkless.tdlib`.
- **Native Binaries:** `libtdjni.so` must be built from TDLib v1.8.0 source and placed in `app/src/main/jniLibs/[ABI]/`.

## Included ABIs
- armeabi-v7a
- arm64-v8a
- x86
- x86_64
